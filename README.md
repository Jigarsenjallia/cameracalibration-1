Building on Mac OS X
--------------------
- - -

	>> brew install opencv
	
	[imagefilters] >> make
	

Building on Ubuntu
------------------
- - -

	>> sudo apt-get install libopencv-dev
	
	[imagefilters] >> make
	
Running
-------
- - -
	
interative mode  

	>> bin/calibrate

batch mode

	>> bin/calibrate filename[s]
